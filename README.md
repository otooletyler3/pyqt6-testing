# PyQt6 Learning Projects

A collection of intuitive PyQt6 application to learn GUI development.

## Projects

- **counter/** - Simple increment/decrement app
- **text-editor/** - Text editor with save/load functionality
- **settings/** - Settings app with default and save functions

## Setup

Create / activate virtual environment (venv):
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install PyQt6
```

Please read each project's README.md for further instructions.
